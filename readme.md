\-Google Vertex $300 Credit

Google provides $300 in credit for all new users to google cloud. You can use these credits by setting up vertex and accessing the latest google models like gemini-3.1-pro

Go to google->readme.md for setup instructions





\-Perplexity Pro

Perplexity previously had an offer which gave free perplexity pro, you can then use claude sonnet through this and the proxy in WIP->perplexity
